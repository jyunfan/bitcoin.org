選擇你的錢包

擁有 Bitcoin 錢包你才能進行交易。錢包提供你用來接受其他人匯款的 Bitcoin 位址，也讓你把錢匯出。用法跟 email 類似，在離線時你仍會收到匯入的 Bitcoin，所有的錢包也都彼此相容。另外，在開始使用 Bitcoin 之前，請務必先看過Bitcoin 使用須知。

快速上手指南

如果你是 Bitcoin 新手，這些錢包是你好的開始。

  • Bitcoin-Qt 提供 Windows、Mac、Linux 版本的 App 下載。
  • Bitcoin Wallet 提供 Android 版本，可在手機或平板執行。

成為 Bitcoin 網路

你有臺從不關機並連上網路的電腦嗎？你可以在上面執行 原生 Bitcoin 客戶端 來幫助這個社群。這個原生客戶端在初次執行時會消秏較多電腦資源並花上一整天來同步，但完成後你的電腦就能協助 Bitcoin 網路檢查並接替交易。

錢包軟體

在你的電腦上安裝錢包軟體，它能讓你完全控制你的錢包。你必須負責備份自己的 Bitcoin 並像現金一樣保護它。

Bitcoin-Qt

Bitcoin-Qt 是原生 Bitcoin 客戶端，它也是整個網路的骨幹。它提供最高等級的安全性、隱密性、及穩定性，但是它的功能比較少，同時會需要許多空間和記憶體。

Multibit

Multibit 是一個輕量級的客戶端，著重在快速且容易上手。它會與網路同步，幾分鐘內就可以使用。另外它支持多種語言。它是非技術背景使用者的好選擇。

Armory

Armory 是執行於 Bitcoin-Qt 之上一個先進的客戶端，為 Bitcoun 專業使用者擴充許多功能。它有提供多種備份、加密功能，也允許將 secure cold-storage 放在離線的電腦。

Electrum

Electrum 注重速度和簡易性，只佔用少量的資源。它將 Bitcoin 系統裡最複雜的計算留給遠端的伺服器執行，另外它提供使用者透過密碼還原錢包的功能。

Mobile wallets

Mobile wallets 讓你把 Bitcoin 放口袋裡帶著走。利用掃瞄 QR code 或 NFC 的 "tap to pay" 功能，就可以交換 coins 和在實體商店消費。

Bitcoin Wallet

Bitcoin wallet 是輕量的行動客戶端，支持 Android 和 BlackBerry OS 兩種版本。這個客戶端不需和任何線上服務綁定就可以運作。它相容 QR code 掃瞄及 NFC。

Blockchain.info

Blockchain.info is a user-friendly hybrid wallet. It stores an encrypted version of your wallet online but decryption happens in your browser. For security reasons, you should always use the browser extension and email backups.

Blockchain.info 是一個對使用者友善的 hybrid 錢包。它將錢包的加密版本儲存在線上，但在你的瀏覽器裡解密。為安全起見，你應該使用它的瀏覽器擴充套件，同時用 email 來備份。
