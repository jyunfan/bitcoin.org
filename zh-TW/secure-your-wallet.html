保護你的錢包

你的 Bitcoin 如同實體錢包需要被保護。Bitcoin 提供史上最簡便的跨國匯款功能，但對安全性的顧慮也隨之而來。只要使用方法正確，Bitcoin 能提供相當高等級的保障。為了保護自己的財產，你必須學會良好的使用習慣。以下是你該注意的事情。

Be careful with online wallets

謹慎使用網路錢包

網路錢包的服務和一般網路銀行很像。小心地選擇服務提供者，你要靠它們來保護你的財產，而你要負責記住和保護自己的密碼。直至今日網路錢包還未能提供像實體銀行同等級的保障。使用 two-factor 認證或類似的功能可以進一步提升帳號安全性。

備份錢包

Bitcoin 服務及軟體允許你用列印在紙上或存在隨身碟的方式備份自己的錢包。妥當保管你的備份，在電腦故障或操作錯誤的時候就要用到它了。

備份整個錢包

為了保護隱私，你的錢包使用許多 private key 來處理交易造成的變動。只對可見的 private key 備份並不足以無法回復你絕大部分的財產。

對線上備份加密

線上備份很容易遭竊，電腦只要連上網路就面臨許多惡意軟體的威脅。對所有曝露在網路上的備份加密是提高安全性的重要步驟。

使用異地備援

單點備份失常對安全性是件壞事。如果你的備份有一個以上的保存方式，就能降低遇上無法用備份還原錢包的倒楣事。你應該考慮使用 USB 隨身碟、輸出成實體文件、燒成光碟等多種備份方式。

定期備份

雖然所有的 App 都將支持使用單次備份還原錢包的功能, 仍建議你要定期備份錢包以確保近期內新增或修改的所有 Bitcoin 位址都有存到備份。

對錢包加密

你可以設定從 Bitcoin 錢包裡拿錢時需要輸入密碼。密碼在面臨小偷和駭客入侵時提供了多一層保障，但你仍然要提防壞人透過軟硬體側錄打字內容盜取它。

不要忘記密碼

如果忘記密碼，你將永遠拿不回 Bitcoin 上的財產。Bitcoin 跟銀行不同，它沒有重置密碼的功能。最好確定長時間沒使用的情況下你還能記得密碼，如果沒有信心，你也許要把密碼寫下來鎖在保險箱之類安全的地方。

確定密碼不會被猜中

同時使用字母、數字、特殊符號，並且長度大於十六個字的密碼才有足夠的安全性。單純字母或有意義單字組合的密碼很容易被破解。當然，難記不該成為忘記密碼的理由。

使用離線錢包保管財產

離線錢包 (或稱 cold storage) 提供最安全的保管方式。它是指將錢包方在一個沒有網路連線且安全的地方。如果規畫的好，它可以對各種電腦弱點提供相當的保障。在你用它來做大筆交易之前，確定離線環境的設置有經過完整的測試。離線錢包搭配備份與加密是一個好的使用方式。下面簡介一些可行的方法。

離線交易認證

這個作法使用兩台電腦共享同一個錢包內的部分資訊。其中一台處於離線狀態，它是唯一存有完整錢包並能認證交易的機器。另一台與網路連線，它只擁有一個用來產生未認登交易用的可見的錢包。你按照下列步驟來產生安全的交易：

  1. 在連線的電腦上產生新交易，將交易存到 USB 随身碟。
  2. 在離線電腦上認證該交易。
  3. 把認證完成的交易回傳有連線的電腦。

因為與網路連線的電腦無法對交易進行認證，可以防止它受到入侵後被提取財產。Armory可以用來做離線交易認證。


暫存設備

這個作法是將錢包載入一個暫存的設備上。譬如用 Linux live CD 開機，透過 USB 隨身碟把 light SPV 錢包軟體及設定載入並執行交易。使用可信任的唯讀設備並只將資料載入記憶體的方式開機，惡意程式就無法入侵，同時你的錢包不會在硬碟上留下記錄。但下面有些你要特別留心的事項。

財產消失

暫存的方式可能讓財產永遠消失。如果你的錢包沒有正確地從外接的永久性儲存裝置 (譬如 USB 隨身碟) 載入的話，你對錢包進行的變動就會永遠消失。其中包含用來接收最後一筆交易變動而新增的 Bitcoin 位址。

密碼錯誤

在暫存設備上使用的鍵盤配置可能與實際鍵盤不同，造成輸入意外的字元。在有加密的情況，會讓你輸入錯誤的密碼。你可能要先在螢幕上輸入看看內容是否如你所想。

不留記錄

只要電腦有連接硬碟或其他儲存裝置，就仍有留下 private key 記錄的風險。在載入錢包前，你可能要將卸下所有硬碟，或停止 swap 分割。

多重認證以防竊

Bitcoin 的多重認證功能要求使用一組以上 private key 簽章才能進行交易。目前來看只對 IT 領域的使用者可行，但預期未來可以提供更多使用這個功能的方式。多重認證讓組織或團體的成員，在經過多數人認證交易後從它的帳戶內支出。它也可以讓未來的網路錢包提供多重認證位址，小偷必須同時入侵你的電腦和網路錢包的伺服器才能竊取財產。

行動裝置小額付款

在手機上的 Bitcoin 錢包就像真的皮夾一樣可以裝錢。Bitcoin 錢包考慮到你不會想沒事帶著大筆鈔票逛街，提供類似的額度限制機制，但你可以隨時儲值行動裝置，這樣一來兼顧了安全與便利。

交代遺囑

當你離開人世後，如果沒有其他人知道你錢包存放的地點或密碼的話，裡頭的財產將永遠無法被取出。你必須花時間想想如何處理這件事。
