參與開發

Bitcoin 是開源的，故任何開發者都能貢獻此專案。你所需要的一切都在Github repository上。請確實閱讀並遵守 README 裡的開發流程，提供高品質的程式碼，以及尊重所有的指引。

開發的討論位於 github 以及 sourceforge 的 bitcoin-development 郵件群組。較不正式的開發討論在 irc.freenode.net #bitcoin-dev (→web interface; logs)。
